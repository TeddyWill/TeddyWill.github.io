命令行的用法 

今天学习了前端教程第四章命令行基础，其中有一些常用命令如下;
1.ls 表示查看当前文件夹里面的所有文件和文件夹；
2.cat 查看文件内容；
3.mv 重命名文件
4.touch 创建文件
对于一些无法知晓意思的命令，可以前往explainshell.com
然后在输入框输入想查询的命令回车，既可以得到该命令的解释

关于git命令有一些使用方法如下：
1.git init 初始化本地仓库
2.git add 将变动加到暂存区
3.git commit -v  提交时显示所有diff信息
